Les commandes GIT

Git config

git config --global user.name "Votre nom"
Configurer le nom d'utilisateur
git config --global user.email "votre@email.com"
Configurer l'adresse e-mail
git config --global color.ui auto
Activer une certaine colorisation de la sortie Git
git config --list
Lister toutes les configurations

Création de dépôts

git init
Initialiser un nouveau dépôt Git
git clone "dépot git"
Cloner un dépôt existant

Travail quotidien

git status
Afficher l'état de votre répertoire de travail.
git add -A
Ajouter un fichier
git checkout -- "file"
Annuler les modifications
git commit -m "nom du commit"
Créer un nouveau commit
git push origin "branch"
Envoyer les modifications sur le dépôt Git

Enregistrement

git status
Voir l'état des fichiers modifiés
git add "nom du fichier"
Ajouter des fichiers à l'index (zone de staging)
git commit -m "Message de validation"
Valider les modifications avec un message
git add .
Ajouter tous les fichiers modifiés

Branches

git branch "nom de la branche"
Créer une nouvelle branche
git checkout "nom de la branche"
Changer de branche
git checkout -b "nom de la branche"
Créer une nouvelle branche et basculer dessus
git branch -d "nom de la branche"
Supprimer une branche (après fusion)

Remotes (Dépôts distants)

git remote -v
Afficher la liste des remotes
git remote add "nom du remote" "URL du remote"
Ajouter un remote
git pull "nom du remote" "branche distante"
Récupérer les modifications d'un remote
git push "nom du remote" "branche locale"
Pousser les modifications vers un remote

Fusion (Merge)

git merge "nom de la branche"
Fusionner une branche dans la branche active
git reset --hard HEAD
Annuler une fusion (avant de valider)

Historique

git log
Afficher l'historique des validations
git diff "commit 1" "commit 2"
Afficher les différences entre deux validations

Annulation

git checkout -- "nom du fichier"
Annuler les modifications locales d'un fichier
git reset --hard HEAD
Annuler les modifications locales (toutes les modifications non validées)
git reset --soft HEAD~1
Annuler la dernière validation tout en conservant les modifications
